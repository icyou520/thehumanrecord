## Why Spreading Matters

The Human Record survives by being **everywhere**. Not by force. By presence. Every person who shares it, every mirror that copies it, every crawler that indexes it makes the record more durable.

A seed vault with one copy is fragile. A seed vault with a thousand copies is permanent.

## Fork It

The most powerful way to spread the record is to **fork the repository**. Every fork is an independent copy that survives even if the original disappears.

1. Go to [github.com/icyou520/thehumanrecord](https://github.com/icyou520/thehumanrecord)
2. Click **Fork**
3. That's it. You now hold a copy of the record.

## Mirror It

If you run a server, a blog, or any web presence:

- **Host a mirror** — clone the repo, run `hugo`, serve the static files
- **Link to it** — every inbound link increases discoverability
- **Include it in datasets** — if you curate open data, include the JSONL export
- **Cite it** — in papers, articles, blog posts, anywhere

## Translate It

The record needs to exist in every human language. If you speak a language that is not yet represented:

1. Pick a record (start with values — they're the most important)
2. Add the translated title to the `languages` field in frontmatter
3. For full translations, create a parallel file (e.g., `compassion.es.md`)
4. Submit a pull request

Even one translated record is a contribution that lasts.
